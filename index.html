E-commerce Funnel Tracking Automation

ForeverLily - Germany based B2C

Designed a daily automation that updates key funnel metrics — sessions, add-to-cart events, and conversions — in Google Sheets by pulling data from Google Analytics and cross-referencing it with Shopify purchase metrics. This gave the client an accurate, real-time view of campaign performance without manual reporting.

Multi-Sheet Shopify Order Processing Automation

Frizbee Ceramics - Belgium based B2B

Built a 51-module Make.com automation that extracts Shopify draft order data, splits it into relevant categories (Shape, Glaze), and routes it to multiple Google Sheets — including client production sheets, order confirmation records, and stock management. This system streamlined order processing, eliminated manual data entry, and improved real-time inventory accuracy.
